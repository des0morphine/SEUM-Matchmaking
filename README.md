### SEUM MULTIPLAYER AND RANKED ELO DUELS ARE NOW IN OPEN BETA

SEUM: Speedrunners from Hell finally has a real multiplayer system with both competitive ranked play and casual modes, inspired by platforms like FACEIT and featuring an ELO ranking system. Track your stats, ranks, match history, and leaderboards at https://seum.online/elo

Play ranked duels, gain or lose ELO, climb through tiers, track your stats, match history, winrate, streaks, and compete in seasonal leaderboards. Or just casually play multiplayer with your friends - you will be able to see each other’s ghosts on any level while playing together in parties of up to 16 players.

## Installation and Matchmaking Guide

### 1. Installation Steps
1. Navigate to the Releases page of this repository and download the latest `.zip` archive.
2. Extract the contents of the archive directly into your main SEUM game directory.
   * Default Steam directory path: `\Steam\steamapps\common\SEUM Speedrunners from Hell`
3. Ensure that `winhttp.dll` and the `BepInEx` folder are placed in the same directory as the game's primary executable file (`.exe`).

### 2. How to Start Playing
1. Launch the game.
2. Open a web browser and log into the platform using your Steam account at https://seum.online/elo.
3. Click the "Find Match" button on the website dashboard.
4. Complete the map ban phase. The system will automatically handle lobby creation and connection.
